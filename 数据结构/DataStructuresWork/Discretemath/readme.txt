共两个程序源文件：DiscreteMath_1.cpp、DiscreteMath_2.cpp
分别对应：最邻近法、最小生成树法

图的输入采用文件读取的方式，两个程序均通过对matrix.txt的读取进行数据输入。
所以两个源文件$$$均可直接运行$$$得出结果，无需在控制台输入
如需更改数据，请直接修改matrix.txt。

默认以输入的第一个顶点为始点，如需更改，直接修改源文件的宏定义，如下
#define START 0 	//从第0个顶点出发

matrix.txt格式
第一行为顶点个数m
接下来m行中，每行均有m个数字，其对应一个m*m的邻接矩阵。
约定：无穷大为99999。
